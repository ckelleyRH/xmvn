XMvn Tutorials

How to embed XMvn in your application

This short tutorial shows how you can use services offered by XMvn from your application. Provided examples guide how to use local artifact resolver service that can find artifacts installed locally in the system.

How to create an extension for XMvn

This tutorial shows how to extend functionality provided by XMvn. It provided examples guiding how to create a very simple extension that allows XMvn to be used to install Eclipse artifacts.
